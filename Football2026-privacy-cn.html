隐私政策

生效日期：2026年6月12日

Football 2026 - 赛事助手（以下简称"本应用"）致力于保护您的隐私。本政策说明我们收集哪些信息以及如何使用这些信息。

信息收集

本应用不会收集、存储或传输任何个人信息。具体而言：

  • 无需注册账号
  • 不收集个人身份信息（姓名、邮箱、电话）
  • 不访问位置数据
  • 不访问照片、通讯录或健康数据
  • 不使用广告标识符或任何追踪技术

本地数据存储

本应用仅在您的设备本地存储以下数据：

  • 用户偏好设置（昵称、关注球队、时区设置、深色模式偏好）
  • 虚拟竞猜数据（虚拟金币余额、投注历史、排行榜分数）
  • 通知偏好设置

这些数据使用 iOS UserDefaults 存储，永远不会离开您的设备，也不会传输到任何服务器。

网络请求

本应用通过 HTTPS 请求访问公共体育数据 API，获取赛程、实时比分、球队信息和球员数据。这些请求不包含任何个人信息。访问的 API 包括：

  • TheSportsDB（公共体育数据库）
  • ESPN 公共体育 API
  • Football-Data.org（可选，使用用户自行提供的 API 密钥）

本地通知

如果您授予权限，本应用可能发送本地通知（如比赛开始提醒、进球通知）。通知调度完全在设备端处理，不涉及任何推送通知服务器。

儿童隐私

本应用不会故意收集13岁以下儿童的信息。虚拟竞猜功能仅使用虚拟货币，不涉及任何真实货币价值。

第三方服务

本应用未集成任何第三方分析、广告或追踪 SDK。

政策变更

我们可能不时更新本隐私政策。任何变更将在上方"生效日期"中反映。
