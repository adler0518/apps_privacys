Privacy Policy · Last updated June 2, 2026

Our Privacy Promise

Call Again is built with privacy as the foundation. We collect no personal data, access no contacts, and make no network requests.


✓ No contacts access ✓ No internet connection ✓ No data uploaded ✓ On-device only

What Data We Collect

None. Call Again does not collect, store, or transmit any personal data to external servers.

The only data stored is what you explicitly enter (phone numbers and contact names for Favorites), and it is stored exclusively in your device's local SwiftData database. This data never leaves your device.

Phone Calls

Call Again uses the iOS tel:// URL scheme to initiate phone calls. This is handled entirely by iOS's native Phone app. Call Again does not log, record, or transmit call history.

Notifications

If you set a call reminder, Call Again schedules a local notification using UNUserNotificationCenter. Notifications are:

  • Scheduled locally on your device
  • Never transmitted to any server
  • Fully under your control — you can disable them at any time in iOS Settings

No Third-Party SDKs

Call Again contains no analytics SDKs, no advertising frameworks, no crash-reporting services, and no tracking libraries of any kind.

Permissions Requested

  • Notifications — Required only if you use the call reminder feature. You can deny this permission; the rest of the app works without it.

We do NOT request access to Contacts, Microphone, Location, Camera, or any other sensitive system resource.

Children's Privacy

Call Again does not collect any data from any user, including users under the age of 13. The app is safe for all ages.

Changes to This Policy

If we ever update this policy, the revised version will be posted at this URL. For a purely offline, no-data app like Call Again, we do not anticipate significant changes.
